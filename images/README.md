## Image specs
- **Format**: `.jpg` for photos (smaller), `.png` if you need transparency
- **Size**: 1200–1600px on the longest side is plenty; the CSS resizes for display
- **Compression**: aim for under 300 KB per file (use tinypng.com or squoosh.app)
- **Aspect ratio**: portrait (3:4 or 4:5) works best for service/bridal cards; landscape (16:9, 21:9) for hero banners

If an image is missing the page still loads — the browser shows the alt text in
an empty box. Just drop in the file and refresh.
